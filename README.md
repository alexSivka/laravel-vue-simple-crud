# Simple crud laravel+vue

demo http://gtn-info.ru/jobs

## install

``` bash
# clone the repo
$ git clone https://github.com/alexSivka/laravel-vue-simple-crud my-project

$ cd my-project

```
database options in .env
```bash

$ composer install

$ npm install

$ php artisan key:generate

$ php artisan migrate:refresh --seed

$ npm run dev

$ npm run dev

```
